## **StackHouse Ordering System** 

## **Overview** 

StackHouse is a Java-based deli ordering system that allows customers to create and customize sandwich orders through both a console application and a graphical user interface (GUI). The project was built using object-oriented programming principles and includes features such as order customization, signature sandwiches, receipt generation, and input validation. 

## **Features** 

- Create custom sandwiches 

- Order signature sandwiches:  BLT, Philly Cheese Steak 

- Add drinks and chips 

- Customize toppings and sauces 

- Remove toppings from signature sandwiches 

- GUI built with Java Swing 

- Receipt generation and file saving 

- Input validation and error handling 

## **Technologies Used** 

- Java 

- Java Swing 

- Object-Oriented Programming (OOP) 

- File Handling
